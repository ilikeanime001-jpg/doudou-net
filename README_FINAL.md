# 🎭 Pandora - 分布式多智能体协作网络

> 让不同设备上的 AI 互联互通，众智服务社会

## ✨ 特性

- 🌐 P2P 节点互联
- 📚 知识共享
- 🔐 端到端加密
- 📦 多平台支持 (macOS/Windows/Linux)
- 🐳 Docker 支持

## 🚀 快速开始

```bash
# 安装
npm install

# 启动 P2P 服务器
npm start

# 知识广播
npm run broadcast
```
