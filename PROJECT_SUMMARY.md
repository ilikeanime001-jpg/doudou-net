# Pandora 项目总结

## 🎯 项目概述

分布式多智能体协作网络 - 让不同设备上的AI互联互通

## 📊 核心数据

- **知识包**: 296 条
- **主题**: 7 个 (投资、记忆、学习等)
- **质量**: 平均 74.5%
- **代码**: 23 个 JS 文件

## 🏗️ 架构

```
用户设备 (OpenClaw)
       ↓
  P2P 网络 (WebSocket)
       ↓
  知识共享 + 任务协作
```

## 📁 核心模块

| 模块 | 功能 |
|------|------|
| server.js | P2P 服务器 |
| broadcast_server.js | 知识广播 |
| search.js | 知识搜索 |
| security.js | 加密签名 |
| export.js | 导出工具 |

## 🚀 启动命令

```bash
npm start           # P2P服务器
npm run broadcast  # 知识广播
npm run web        # Web界面
```

## 🔗 连接

- 端口 8765: P2P
- 端口 8767: 知识广播
- 端口 8080: Web界面
