# Pandora 使用教程

## 快速开始

### 1. Mac 作为服务器分享知识

```bash
cd pandora

# 首次: 扫描知识库
npm run knowledge

# 启动知识广播服务器
npm run broadcast
```

服务器会显示:
```
🎭 Pandora 知识广播服务器
端口: 8765
知识: 296 个
```

### 2. Windows 接收知识

```powershell
cd pandora
node src/search.js
```

操作:
1. 输入 `4` 连接服务器
2. 输入 Mac 的 IP: `192.168.1.156:8765`
3. 等待接收知识
4. 输入 `2` 查看已接收知识

### 3. 搜索知识

接收完知识后:
1. 输入 `1` 搜索
2. 输入关键词，如: `投资`、`现金流`、`比亚迪`

## 知识主题

- **投资** - 股票、投资框架 (216条)
- **记忆** - 每日记忆 (25条)
- **学习** - 学习笔记 (16条)
- **偏好** - 用户偏好设置
- **目标** - 目标和计划
- **操作** - 操作指南
- **general** - 其他知识

## 高级用法

### 使用 P2P 模式 (双向连接)

```bash
# Mac: 启动 P2P 服务器
npm start

# Windows: 连接
node src/client.js ws://192.168.1.156:8765
```

### 自定义端口

```bash
npm run broadcast -- --port 8766
```

## 故障排除

| 问题 | 解决方法 |
|------|----------|
| 连接失败 | 检查 IP 地址是否正确 |
| 知识为0 | 确保服务器先启动 |
| 搜索不到 | 先连接服务器接收知识 |
