# DoudouNet 示例代码 | Examples

## 快速开始 | Quick Start

### 运行服务器 | Run Server

```bash
node examples/simple-server.js
# 默认端口: 8765
# 自定义端口: node examples/simple-server.js --port 8766
```

### 运行客户端 | Run Client

```bash
# 连接到本地服务器
node examples/simple-client.js

# 连接到远程服务器
node examples/simple-client.js ws://192.168.1.100:8765
```
